# Minitsuke

**Minitsuke** is an AI-powered Japanese Learning Ecosystem designed to help Thai learners acquire Japanese as a lifelong skill.

> Learn Japanese until it becomes part of you.

## Project Architecture Name

**JLE — Japanese Learning Ecosystem**

## Core Philosophy

Understand → Practice → Apply → Master

## Development Policy

- Browser First
- Free Tier First
- Documentation as Code
- Content First
- CMS Driven
- AI Native
- Single Source of Truth
- Modular Architecture
- Accessibility First
- Security by Design
