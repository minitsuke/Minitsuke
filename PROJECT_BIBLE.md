# Minitsuke Project Bible v1.0

## Brand Name

**Minitsuke**

Derived from Japanese **身につける (mi ni tsukeru)**, meaning to acquire something until it becomes a skill that stays with you.

## Architecture Name

**Japanese Learning Ecosystem (JLE)**

## Mission

Help Thai learners acquire Japanese as a real-life skill through structured learning, practice, review, and AI-assisted guidance.

## Vision

Build the most trusted Japanese learning ecosystem for Thai learners, expandable to global learners in the future.

## Core Values

- Learner First
- Learning by Understanding
- Quality Over Speed
- AI with Responsibility
- Content First
- Simplicity
- Continuous Improvement
- Accessibility for Everyone

## Non-Negotiables

- No hardcoded learning content
- All learning content must be CMS-ready
- Browser-first development
- Free-tier-first tooling
- Documentation as Code
- Single Source of Truth
- Multilingual-ready
- Security by Design
- Accessibility First

## Product Pillars

1. Learning
2. Knowledge
3. AI
4. Assessment
5. Review
6. Analytics
7. CMS
8. Platform
